# PySnake

## Getting Started

1. **Prerequisites:**
   - Install Python: [https://www.python.org/downloads/](https://www.python.org/downloads/)
   - Install Pygame: Open your terminal and run `pip install pygame`

2. **Clone and Run:**
   ```bash
   git clone https://github.com/pe4nutlord/PySnake.git
   cd pysnake
   python snake.py
